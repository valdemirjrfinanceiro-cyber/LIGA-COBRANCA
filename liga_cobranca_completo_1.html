📖 Passo a Passo — Como Configurar
1
Criar conta Google

Acesse accounts.google.com e crie uma conta Gmail para o Grupo Estillo (ex: financeiro@grupoestillo.com). Essa conta vai guardar todos os comprovantes.

2
Criar projeto no Google Cloud

Acesse console.cloud.google.com → clique em "Novo Projeto" → dê o nome "Liga Cobrança Estillo" → clique em Criar.

3
Ativar Google Drive API

No menu lateral → "APIs e Serviços" → "Biblioteca" → pesquise "Google Drive API" → clique em Ativar.

4
Criar credenciais

"APIs e Serviços" → "Credenciais" → "Criar credenciais" → escolha ID do cliente OAuth 2.0. Tipo: Aplicativo da Web. Em "Origens JavaScript autorizadas" adicione o endereço onde o sistema estiver hospedado (ou http://localhost para testes).

5
Copiar o Client ID e API Key

Após criar, copie o Client ID e cole no campo acima. Para a API Key: "Criar credenciais" → "Chave de API" → copie e cole acima.

6
Criar pasta no Google Drive

Abra o Google Drive com a conta criada → clique em "Nova pasta" → nomeie "Comprovantes Estillo" → abra a pasta → copie o ID da URL:

drive.google.com/drive/folders/ESTE_É_O_ID

Cole esse ID no campo "ID da Pasta" acima.

7
Testar e compartilhar o link

Clique em "Testar Conexão" acima. Se der ✅, o sistema está pronto. Copie o link desta página e mande no grupo do WhatsApp para os vendedores acessarem.
